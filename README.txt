USB Mini-B pads
---------------

 /--------------------------------------------------\
 |                                                  |
U| = 1 red (+5V)                                    |
S| = 2 white (Data-)                                |
B| = 3 green (Data+)                                |
|| = 4 ------                                       |
B| = 5 black (GND)                                  |
 |                                                  |
 \--------------------------------------------------/
